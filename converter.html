EdAi · setup tool

Spreadsheet → JSON

Convert your .xlsx playbook into the file EdAi loads automatically. Runs entirely in your browser — nothing is uploaded.


  


  

    
1
Column A — the phrasings people might type. Many rows can feed one answer.

    
2
Column B — the answer. A new value in B starts a new topic; leave B blank to keep adding phrasings to the topic above.

    
3
Column C — follow-up options (one per row). Put every option in its own C cell.

    
4
Column D — the reply for each option. Leave blank to use EdAi's default.

  


  
After downloading, put knowledge.json in the same folder as index.html. Re-run this tool whenever the spreadsheet changes.
